# Octree Basics

An `Octree` is a data structure used for efficient spatial querying. It provides a method for scaling up [traversal loops](../hosting-3d-model/bpy_with_lod.md) from O(n) runtime to O(log n) runtime. For large scenes, the octree is an almost fundamental aspect of performance.
