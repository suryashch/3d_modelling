# OBJ Files Exploratory Data Analysis

OBJ files have been referred to as "The CSV of 3D geometry". In this analysis, we explore the structure of an OBJ file, doing some Exploratory Data Analysis (EDA), and seeing if we discover any nuances. The OBJ file I shall be using the same as before `human-foot-base-mesh` (@human-foot-base-mesh).
